A Revolutionary Website Building Tool!

What is Bootstrap?

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

What Makes Bootstrap Amazing?

Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Bootstrap's Wonderful Components!

Typography!

Bootstrap has got you covered with all of your text needs! Use the classes to edit the size, display, and decoration for your headings and paragraphs.

Colours!

This open source toolkit has an excellent selection of colour classes. You can edit the colour of almost any component on the page.

Alerts!

Bootstrap allows you to quickly edit in alerts for your website. Perfect for bringing attention to important details that might have been missed otherwise.

Navs!

This service has preset navigation features that will help organize your links in no time! Not only that, but you can use navbars for other links located on the page.

How to Use Bootstrap in 3 Easy Steps!

Warning: These steps might not be easy to everyone and will require time and patience to fully understand.

Step One

Download Bootstrap by clicking the button above OR click on this link here to access the bootstrap template. Copy and paste this template in your html to gain access to bootstraps features!

Step Two

Remember that Bootstrap focuses on mobile design first! After understanding the layout for your smallest viewport design, you can work up to tablet and desktop layouts.

Step Three

Start adding components and content to your page with using Bootstrap classes! No need to worry about styling in CSS as you can give HTML elements specific classes to chnage its properties. It's easy to use and great for clean website structure!
